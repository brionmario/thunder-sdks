# Organization Components

Organization components display and manage the authenticated user's organizations. These are intended for B2B applications where users belong to one or more organizations.

---

## `ThunderOrganization`

Displays the name of the organization the current session is scoped to.

### Usage

```kotlin
import io.thunder.compose.components.presentation.organization.ThunderOrganization

ThunderOrganization()
```

### Base Variant

`BaseThunderOrganization` provides the `Organization?` to a custom content lambda:

```kotlin
BaseThunderOrganization { org ->
    Text(org?.name ?: "Personal")
}
```

## `ThunderOrganizationList`

Lists all organizations the authenticated user belongs to.

### Usage

```kotlin
import io.thunder.compose.components.presentation.organization.ThunderOrganizationList

ThunderOrganizationList(
    onSelect = { org -> navController.navigate("org/${org.id}") }
)
```

| `modifier` | `Modifier` | âťŚ | Compose modifier |
| `onSelect` | `((Organization) -> Unit)?` | âťŚ | Called when the user taps an organization |

### Base Variant

`BaseThunderOrganizationList` exposes `(organizations, isLoading, error, onSelect)` to a content lambda:

```kotlin
BaseThunderOrganizationList { orgs, isLoading, error, select ->
    if (isLoading) CircularProgressIndicator()
    else LazyColumn {
        items(orgs) { org ->
            Text(org.name, modifier = Modifier.clickable { select(org) })
        }
    }
}
```

## `ThunderOrganizationSwitcher`

Picker to switch the active session to a different organization via token exchange.

### Usage

```kotlin
import io.thunder.compose.components.presentation.organization.ThunderOrganizationSwitcher

ThunderOrganizationSwitcher()
```

### Base Variant

`BaseThunderOrganizationSwitcher` exposes `(organizations, currentOrg, isSwitching, error, switchOrg)`:

```kotlin
BaseThunderOrganizationSwitcher { orgs, current, isSwitching, error, switchOrg ->
    Column {
        error?.let { Text(it, color = Color.Red) }
        orgs.forEach { org ->
            Text(
                org.name,
                modifier = Modifier.clickable(enabled = !isSwitching) { switchOrg(org) }
            )
        }
    }
}
```

## `ThunderLanguageSwitcher`

Picker to switch the active UI locale for SDK component labels.

### Usage

```kotlin
import io.thunder.compose.components.presentation.organization.ThunderLanguageSwitcher

ThunderLanguageSwitcher()
```

### Base Variant

`BaseThunderLanguageSwitcher` exposes `(locales, activeLocale, setLocale)`:

```kotlin
BaseThunderLanguageSwitcher { locales, active, setLocale ->
    locales.forEach { locale ->
        Text(
            locale,
            fontWeight = if (locale == active) FontWeight.Bold else FontWeight.Normal,
            modifier = Modifier.clickable { setLocale(locale) }
        )
    }
}
```
